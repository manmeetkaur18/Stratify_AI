# Stratify AI Frontend

This is the frontend for Stratify AI, built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui.

## Features

- startup dashboard with live intelligence metrics
- decision playground with sliders and instant simulations
- competitor comparison view
- scenario comparison table
- settings page for startup profile inputs

## Run

```powershell
npm install
npm run dev
```

If needed, add a local env file:

```env
VITE_API_URL=http://127.0.0.1:8000
```
